# Academic-Research

This research focuses on the application of machine learning for clasification of lithology from geophysical well logs using datesets from the Force 2020 Machine learning competition in October 2020.

## Links
- Competition/github page: [Visit link](https://github.com/bolgebrygg/Force-2020-Machine-Learning-competition)
- Competition Winner's blog post: [Visit link](https://ibrahim-olawale13.medium.com/force-2020-machine-learning-lithology-predictionwinning-solution-8cbf78290b41)


## Data Source
Lithofacies data used in this project was provided by the FORCE Machine Learning competition with well logs and seismic data (Bormann et al., 2020).

**Citation:**
Bormann P., Aursand P., Dilib F., Dischington P., Manral S. 2020. 2020 FORCE Machine Learning Contest.
